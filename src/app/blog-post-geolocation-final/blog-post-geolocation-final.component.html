Whats so great?

Random location generator

Okay, its not really random...but it gets the job done! The point behind this feature is to generate locations on the map that you would need to go to complete that 'mission'. What I did to do this was basically each time the "Generate Random Point" button is clicked a number is randomly generated and if that number is even, it adds to the latitude and longitude, if it is odd then it subtracts from the latitude and longitude.
