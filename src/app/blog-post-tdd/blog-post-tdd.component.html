The Challenge

Some positive integers n have the property that the sum [ n + reverse(n) ] consists entirely of odd (decimal) digits.
For instance, 36 + 63 = 99 and 409 + 904 = 1313. We will call such numbers reversible; so 36, 63, 409, and 904 are reversible. Leading zeroes are not allowed in either n or reverse(n).
There are 120 reversible numbers below one-thousand.
How many reversible numbers are there below one-billion (109)?

Test Driven Development

TDD has many benefits. I have chose to follow TDD as close as possible because I have found that it makes code more enjoyable, lays out requirements clearly, creates a sort of contract for me to meet, allows me to gauge my work better. Some other benefits of TDD include:
  • TDD fits right in with Agile customer-focused teams because the requirements become the tests
  • Speeds up delivery becuase feedback is received promptly
  • Encourages developers to create small modularized methods. Keeping code clean and precise
  • Tests can be run automatically within a few seconds.
  • Gives developers a sense of confidence by completing small tasks throughout the work day
All in all, TDD is very much worth it. It may take you a little bit longer to write the tests, but it will save you that much more (plus some) in the end! GO TDD!!
