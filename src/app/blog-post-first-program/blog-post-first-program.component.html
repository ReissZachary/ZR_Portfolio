Making Your First C# Program

This article will teach you how to write your first program using the C# language.

By the end of this article, you will have created a program that asks for your name, greets you with the name you give it, asks for your birth year, then calculates your age.

Prerequisites

  • Have installed Visual Studio 2019 - Community on your computer
  • Have installed the .NET SDK on your computer

Step 1: Creating A New Project

Open Visual Studio and click "Create a new project"

Then:

  1. In the search bar towards the top of the window type in Console
  2. Choose the Console App (.NET Core) option
  3. Then click Next

Next:

  1. in the Project Name box, enter a name for your project. I named mine MyFirstProgram (no spaces)
  2. Then click Create

Step 3: Run Your Program

Remember that Play Button we mentioned earlier located at the top of the window on the toolbar?

Well, it is finally time to push it! Go ahead, give it a good ol push.

Doing this will Run or Execute the code you just wrote! You should see a small window pop up on your screen. Don't worry, this is good!

This is called the Console (sounds familiar right?). At the top of this window you should see the text, "Hello, welcome to your first application" followed by "Lets begin by entering your name:"

Go ahead! Type in your name and hit enter!

There you have it, you have successfully created your first program using the C# language!
