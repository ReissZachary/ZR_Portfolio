Latitude and Longitude via Geolocation

TL;DR

What: An Angular app that can get you location and displays your lat and long on the screen

Why: To experiment and begin working on an app idea that will rely heavily on Geolocation

How: Using the library @agm/core and Google Maps api

The Big Picture

This application may seem silly and simple but this is a key part of and app that I want to make. To summarize my app idea, it will be a 'hacking' game where you are given intel you need to hack using your phone. You will see your location and your targets location. You are to get close enough to the location of your target without being caught to hack them and get the secret information from them. The target could, in theory, be another person playing, a random generated location, a stationary object, or a moving object. This is why Geolocation is crucial for this app.

So It Begins

AGM

After setting up my Angular app I went ahead and installed @agm/core. AGM stands for Angular Google Maps and has many different tools that will allow us to get our latitude and longitude location. This is installed using the command npm install @agm/core

@NgModule Setup

After installing agm/core we need to import the AgmCoreModule in our app.module.ts file. This will allow our app to utilize the agm core tools. Along with this import I provided my google maps api key to use later when I need to add a map. You can get your google map api key here
